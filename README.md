# Sudhanshu Gochar - Portfolio

A modern, interactive portfolio website showcasing Machine Learning and Web Engineering projects, skills, and experience.

## About

This is a professional portfolio website built with modern web technologies, featuring:

- Machine Learning Projects
- Web Development Projects
- UI/UX Design Projects
- Web Design Projects
- Professional Experience
- Technical Skills
- Achievements
- Contact Information

## Technologies Used

This project is built with:

- **Vite** - Next generation frontend tooling
- **React** - UI library
- **TypeScript** - Type-safe JavaScript
- **Tailwind CSS** - Utility-first CSS framework
- **shadcn/ui** - High-quality React components
- **Framer Motion** - Animation library
- **React Router** - Client-side routing

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

```sh
# Clone the repository
git clone https://github.com/Sudhanshugochar/data-driven-insights.git

# Navigate to the project directory
cd data-driven-insights

# Install dependencies
npm install

# Start the development server
npm run dev
```

The application will open at `http://localhost:8080`

## Development

```sh
# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview

# Run linting
npm run lint
```

## Deployment

### Deploy to Vercel

The easiest way to deploy is using [Vercel](https://vercel.com):

```sh
# Install Vercel CLI
npm i -g vercel

# Deploy
vercel
```

Or connect your GitHub repository to Vercel for automatic deployments on every push.

### Build Output

The production build is located in the `dist/` directory after running:

```sh
npm run build
```
